var is the oldest keyword to write a variable. Using var we can reassign the same variable again not only reassign but also redeclare is also possible with var. Regarding this let (let it be update) keyword is a little bit different. Using let we can't redeclare but reassign. On the other hand const keyword dosen't allow neither reassign nor redeclare. It is very strict in its usage.

Arrow function is a new invention from ES6. It has something new and better than regular function. We can write an arrow function without the keyword function itself. If we want to get return from a regular function we have to write return keyword explicitly, but arrow function has a default return implicitly (only for single line). An arrow looks smarter than regular function.

The map() method do something over an array and return a new array. The forEach() method do something over an array but it dosen't return anything. The filter() method do something conditionally over an array and return array of matched elements. The find() method do something conditionally over an array and return the first matched element.

Template string (literal) is enclosed by backtick (`) characters instead of double or single quotes. Double or single quote allows us to write only string within that, but template string gives us a greater facility regarding this. It allow us to pass parameters, methods, expressions and many more. It adds a new dimension for the developers to create something on the air.
